SigmaTizm
A Virtual Additive Synthesizer
Version 1.01

The MIDI settings dialog lets you change the MIDI input and output devices and their channels. The list of devices is generated based on the MIDI input and output devices available on your system.

The "MIDI Output Enabled" checkbox enables SigmaTizm to operate as a MIDI controller to control external hardware. It is off by default to prevent extra sounds from being generated in addition to SigmaTizm's tone generator because many Windows machines have the Microsoft Wavetable Synthesizer set as the default device.
